Stored Procedure Class Wizard

This tool was contributed by George Poulose

Stored Procedure Class Wizard (SPCW) is a development tool that can be used exclusively to generate class files (.cpp and .h) derived from MFC’s CRecordset for stored procedures. Currently SPCW supports Microsoft SQL Server v 6.5 and Sybase SQL Anywhere v 5.5 and can be extended to any database. The idea is to speed up the development process of client-server projects involving aforementioned packages under Windows NT/95. SPCW is an extension to DevStudio's class wizard.

System Requirements:

VC++ v 5.0, MFC v 4.21, NT 4.0/95

Running the application:

Extract it to a <\temp> folder and then run.
