MFC HTML-Help classes.

This article was contributed by Andreas Dunkel.

As extension for the IE enviroment, Microsoft™ has introduced a new HTML-based help system. This works using HTML help files, including Java and Javascript. The best part is that it work without problems when using large and colorful pictures, including animated gifs.

For help with this system consult the HTML help SDK at http://www.microsoft.com/workshop/author/htmlhelp/

The SDK includes samples, but unfortunately they are not the best, and are written using only straight C.

With my classes you can include the HTML-Help in an easy way for all MFC-Projects. The sample project shows how you can do this. It includes a full HTML-help project with images for your applications (you can delete the old win31 pictures from your help system!) and shows how you can include the help system in a property page (see the accompanying image). The implementation is very simple.
