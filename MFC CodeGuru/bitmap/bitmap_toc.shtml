This section is now being maintained by Raul Ciotescu.

Many of the code snippets have the following bug. I'll update the articles as soon as I get a chance but meantime here are the two things you should be aware of.

  1. Use GlobalLock() and GlobalUnlock() instead of simply assuming the memory handle to be a pointer.
  2. When selecting a bitmap to a DC, save the previous bitmap in a variable and reselect it at the end before releasing the DC.
