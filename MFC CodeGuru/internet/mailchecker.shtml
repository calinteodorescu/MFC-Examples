Mail Checker

This sample was contributed by Asif Rasheed

Here is an implementation of my POP class. It retrieves mail statistics from any POP server. User can define the time duration for rechecking the mail,

The application was written using MFC 4.2 and Visual C++ 5.0,  It uses

  • WinInet class wrapper
  • CTrayIcon class by Chris Maunder (Chris, you did really a nice work).

I am releasing this app into the public domain with no restrictions.
