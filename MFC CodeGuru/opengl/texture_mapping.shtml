Starting Texture Mapping

This article was contributed by Pierre ALLIEZ.

Download project files 707KB

This is the smallest example to map texture on an object. This sample create an OpenGL rendering context in a dialog box and map a texture on a cube. We can manipulate the object with the mouse, and change texture file with a popup menu item. A timer animate a rotation. Texture files are .RAW files (256 x 256 x 8 bits) and can be editable using Paint Shop Pro.
