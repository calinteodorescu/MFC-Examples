Palette editor

Download

The CDialogPalette class permit easy edition of 256 colors ramps. We can also save and load predefined ramps. The ramp editing is made using nodes, ramps are recalculated between every defined nodes. by example : for a single grey scale ramp, only the first and the last nodes are necessary (black and white), the rainbow ramp is defined using a few nodes, in fact in a few mouse events. Adaptation is possible to insert this class in applications which need ramp definition...
